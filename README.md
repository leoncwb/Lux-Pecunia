🏛️ Lux Pecunia
Ex Tenebris ad Lucem Monetariam (Das trevas à luz monetária)

O Lux Pecunia é um ecossistema de inteligência financeira desenvolvido para transformar dados brutos em clareza patrimonial. Utilizando processamento de linguagem natural, a ferramenta automatiza a classificação de gastos e o preenchimento de orçamentos baseados em ritos de organização financeira.

🛠️ Pilares Tecnológicos (A Estrutura)
O Cérebro (Back-end): Python 3.10+ (FastAPI) para processamento assíncrono.

A Lente (IA): Integração com Gemini API para classificação inteligente de transações.

O Canteiro (Infra): Docker & Kubernetes (K8s) para orquestração e escalabilidade.

A Prumo (CI/CD): GitHub Actions para garantir que cada commit esteja em conformidade.

📐 Roadmap de Construção
[x] Grau 1 (Aprendiz): Script básico de identificação de gastos via CLI.

[ ] Grau 2 (Companheiro): Containerização da aplicação e criação de API REST.

[ ] Grau 3 (Mestre): Integração com Google Sheets API para atualização em tempo real da planilha.

[ ] O Grande Oriente: Bot do Telegram para entrada de dados por voz e texto.

🚀 Como Executar (Ambiente Local)
Bash
# Clone o repositório
git clone https://github.com/seu-usuario/lux-pecunia.git

# Construa a imagem Docker (Em breve)
